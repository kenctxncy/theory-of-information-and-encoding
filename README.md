# Лабораторные работы по теории информации и кодирования

Проект содержит реализацию лабораторных работ по теории информации, помехоустойчивому кодированию и кодам Хэмминга.

## Структура проекта

Проект организован как Rust workspace, состоящий из следующих компонентов:

- **`entropy_fx`** - библиотека с функциями для расчета энтропии, пропускной способности каналов и помехоустойчивого кодирования
- **`entropy_ui`** - графическое приложение на базе egui/eframe для визуализации и проведения экспериментов

## Лабораторные работы

### Лабораторные работы 1-3
- Расчет энтропии источников информации
- Анализ каналов связи с помехами
- Расчет пропускной способности каналов
- Анализ скорости передачи информации

### Лабораторные работы 4-6
- Систематические помехоустойчивые коды
- Коды Хэмминга
- Обнаружение и исправление ошибок
- Модифицированные коды Хэмминга для обнаружения двукратных ошибок

## Требования

- Rust 1.91+ (edition 2024)
- Для сборки Windows версии: `cargo-xwin` (см. [BUILD_WINDOWS.md](./BUILD_WINDOWS.md))

## Быстрый старт

### Установка зависимостей

```bash
# Клонировать репозиторий с сабмодулями
git clone --recursive <repository-url>
cd lab1

# Или если репозиторий уже клонирован
git submodule update --init --recursive
```

### Сборка и запуск

```bash
# Сборка в режиме разработки
cargo build

# Запуск приложения
cargo run --package entropy_ui

# Сборка релиза
cargo build --release

# Сборка для Windows (требует cargo-xwin)
./build-windows-release.sh
```

## Сабмодули

Проект использует git submodules для организации кода:

- `entropy_fx` - библиотека функций
- `entropy_ui` - GUI приложение

### Работа с сабмодулями

```bash
# Обновить все сабмодули до последних коммитов
git submodule update --remote

# Обновить сабмодули при клонировании
git clone --recursive <repository-url>
```
