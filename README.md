# Лабораторная работа №2: Симуляция энтропии дискретных сообщений с учетом помех

## Описание

Программа для моделирования передачи информации через канал связи с помехами. Реализует расчет энтропии, взаимной информации и условной энтропии для дискретных сообщений.

## Структура проекта

```
lab1/
├── entropy_fx/          # Библиотека с математическими функциями
│   ├── Cargo.toml
│   └── src/lib.rs
├── entropy_ui/          # GUI приложение
│   ├── Cargo.toml
│   └── src/main.rs
├── Cargo.toml           # Workspace конфигурация
└── README.md
```

## Возможности

- Генерация случайных вероятностей с нормализацией
- Моделирование канала связи с настраиваемыми помехами
- Расчет матриц переходов и совместных вероятностей
- Вычисление энтропии, условной энтропии и взаимной информации
- Графический интерфейс с поддержкой больших матриц
- Сокращенный вид для матриц размером > 16x16

## Алгоритмы

### Генерация вероятностей
Использует случайные числа от 1 до 100 с последующей нормализацией для предотвращения нулевых вероятностей.

### Матрица переходов
Диагональные элементы: `min_threshold + random * (1 - min_threshold)`
Недиагональные элементы: равномерно распределены через `generate_probabilities()`
